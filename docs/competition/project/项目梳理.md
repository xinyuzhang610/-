## 二、项目定位

### 2.1 一句话定义

> 一个面向教育场景的 Web 应用平台，通过**需求引导**帮助小白老师发现教学需求、体验 AI 智能体工具、并引导他们使用智能体平台创建自己的学习工具。

### 2.2 核心价值（重点）

```
┌─────────────────────────────────────────────────────────────┐
│                    核心价值：需求引导                          │
│                                                             │
│  老师是小白 → 不知道自己要什么 → 需要引导                      │
│                                                             │
│  第1层：文理分科（缩小范围）                                   │
│      ↓                                                      │
│  第2层：需求引导（发现真实需求）                                │
│      ↓                                                      │
│  第3层：工具推荐/创建（解决问题）                               │
│      ↓                                                      │
│  第4层：数据看板（看到效果，形成闭环）                          │
│                                                             │
│  这才是评审会认可的价值：                                      │
│  ✅ 有引导流程（不是静态页面）                                 │
│  ✅ 有 AI 交互（DeepSeek API 实际调用）                        │
│  ✅ 有数据存储（MySQL 记录使用数据）                           │
│  ✅ 有业务逻辑（推荐算法、工具管理）                           │
│  ✅ 有前后端分离架构（Vue3 + FastAPI）                         │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 与师大平台的关系

```
师大平台 = 专业版（老师在平台上搭建复杂的工作流和 Agent）
你们的 Web 应用 = 应用层（引导、体验、跳转）

两者不是替代关系，而是互补关系：
  你们的系统负责：引导需求 → 推荐工具 → 体验 → 跳转到平台
  师大平台负责：实际的 Agent 创建和运行

注意：师大平台接入先放一边，预留接口，后续接入
```

### 6.4 数据库设计

```sql
-- 用户表
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role ENUM('teacher', 'student', 'admin') NOT NULL,
    name VARCHAR(50),
    school VARCHAR(100),
    subject VARCHAR(50),       -- 教师的科目
    grade VARCHAR(20),         -- 教师的年级
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 工具表
CREATE TABLE tools (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),       -- 学科：语文、数学、英语等
    icon VARCHAR(50),          -- 图标标识
    prompt_template TEXT NOT NULL,  -- 预设的系统提示词
    interface_config JSON,     -- 界面配置（模板类型、参数定义等）
    creator_id INT,            -- 创建者ID，NULL表示预设工具
    is_preset BOOLEAN DEFAULT FALSE,
    is_public BOOLEAN DEFAULT TRUE,
    share_code VARCHAR(20) UNIQUE,  -- 分享码
    usage_count INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (creator_id) REFERENCES users(id)
);

-- 工具模板表
CREATE TABLE tool_templates (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    description TEXT,
    prompt_template TEXT NOT NULL,  -- 模板的提示词
    default_config JSON,           -- 默认配置
    icon VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 使用记录表
CREATE TABLE usage_logs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,               -- 可为NULL（匿名用户）
    tool_id INT NOT NULL,
    input_text TEXT,
    output_text TEXT,
    session_id VARCHAR(50),    -- 会话ID（同一轮对话）
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (tool_id) REFERENCES tools(id)
);

-- 收藏表
CREATE TABLE favorites (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    tool_id INT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (tool_id) REFERENCES tools(id),
    UNIQUE KEY (user_id, tool_id)
);

-- 推荐规则表
CREATE TABLE recommend_rules (
    id INT PRIMARY KEY AUTO_INCREMENT,
    subject VARCHAR(50),
    need_type VARCHAR(50),     -- 需求类型
    tool_ids JSON,             -- 推荐的工具ID列表
    priority INT DEFAULT 0
);
```

### 6.5 DeepSeek API 调用流程

```
用户输入 → 前端发送请求 → 后端接收
    ↓
后端从数据库读取工具配置（提示词模板）
    ↓
拼接：系统提示词 + 用户输入
    ↓
调用 DeepSeek API（model: dsv4）
    ↓
获取返回内容
    ↓
返回给前端展示

示例：
  系统提示词："你是一个古诗词赏析助手，请用趣味的方式..."
  用户输入："静夜思"
  拼接后发送给 API → 返回赏析内容 → 前端渲染展示
```
