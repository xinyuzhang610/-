## 三、逻辑设计

### 3.1 表结构设计

#### 3.1.1 用户表（users）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 用户名 |
| password_hash | VARCHAR(255) | NOT NULL | 密码哈希 |
| role | ENUM | NOT NULL | 角色（teacher/student/admin） |
| name | VARCHAR(50) | | 姓名 |
| school | VARCHAR(100) | | 学校 |
| subject | VARCHAR(50) | | 科目（教师） |
| grade | VARCHAR(20) | | 年级（教师） |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

#### 3.1.2 工具表（tools）
| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 工具ID |
| name | VARCHAR(100) | NOT NULL | 工具名称 |
| description | TEXT | | 工具描述 |
| category | ENUM | NOT NULL | 分类（文科/理科/通用） |
| subject | VARCHAR(50) | | 学科 |
| icon | VARCHAR(50) | | 图标标识 |
| prompt_template | TEXT | NOT NULL | 提示词模板 |
| interface_config | JSON | | 界面配置 |
| creator_id | INT | | 创建者ID |
| is_preset | BOOLEAN | DEFAULT FALSE | 是否预设 |
| is_public | BOOLEAN | DEFAULT TRUE | 是否公开 |
| share_code | VARCHAR(20) | UNIQUE | 分享码 |
| usage_count | INT | DEFAULT 0 | 使用次数 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
