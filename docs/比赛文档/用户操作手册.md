## 二、教师端操作指南

### 2.1 注册与登录

#### 2.1.1 注册账号
1. 打开系统首页
2. 点击"注册"按钮
3. 填写注册信息
4. 点击"确认注册"

#### 2.1.2 登录系统
1. 打开系统首页
2. 输入用户名和密码
3. 点击"登录"按钮

### 2.2 需求引导

#### 2.2.1 开始引导
1. 登录后进入首页
2. 点击"开始引导"按钮
3. 选择您的教学方向（文科/理科）
4. 选择您的教学需求
5. 系统推荐适合的工具

#### 2.2.2 使用推荐的工具
1. 在推荐列表中选择工具
2. 点击"立即体验"
3. 输入相关内容
4. 获取AI生成的结果

### 2.3 工具管理

#### 2.3.1 查看我的工具
1. 点击左侧菜单"我的工具"
2. 查看工具列表

#### 2.3.2 创建新工具
1. 点击"创建新工具"按钮
2. 选择创建方式（AI对话/平台创建）
3. 填写工具信息
4. 点击"保存"

#### 2.3.3 分享工具
1. 在工具列表中选择工具
2. 点击"分享"按钮
3. 复制分享链接或下载二维码

### 2.4 数据查看

#### 2.4.1 查看数据看板
1. 点击左侧菜单"数据看板"
2. 查看统计数据
3. 查看使用趋势
4. 查看工具排行

## 三、学生端操作指南

### 3.1 进入系统

#### 3.1.1 通过分享链接
1. 老师分享链接
2. 点击链接直接进入工具使用页面
3. 无需登录

#### 3.1.2 直接访问
1. 打开系统首页
2. 点击"学生入口"
3. 进入工具广场

### 3.2 学习引导

#### 3.2.1 开始引导
1. 进入系统后
2. 回答引导问题
3. 系统推荐适合的学习工具

#### 3.2.2 使用推荐的工具
1. 在推荐列表中选择工具
2. 点击"立即使用"
3. 输入学习内容
4. 获取AI生成的结果

### 3.3 AI问答

#### 3.3.1 开始问答
1. 点击"AI问答"菜单
2. 输入问题
3. 点击"发送"
4. 查看AI回答

### 3.4 工具广场

#### 3.4.1 浏览工具
1. 点击"工具广场"菜单
2. 按分类浏览工具
3. 搜索工具

#### 3.4.2 使用工具
1. 选择感兴趣的工具
2. 点击"立即使用"
3. 输入内容
4. 获取结果

### 3.5 学习记录

#### 3.5.1 查看记录
1. 点击"学习记录"菜单
2. 查看使用统计
3. 查看历史记录
