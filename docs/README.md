# 智教通文档中心

这里集中保存智教通的比赛最终材料、项目辅助资料和本地运行指南。产品代码的启动方式请先阅读仓库根目录的 [README](../README.md)。

## 文档导航

- [比赛最终 Word 文档](competition/final/)：提交时以这里的七份最新版文件为准。
- [竞赛原始材料](competition/source/)：比赛通知和报名表等原始文件。
- [竞赛项目辅助资料](competition/project/)：任务手册、PPT 大纲和项目梳理。
- [开发指南](guides/)：环境配置和技术栈说明。

## 资料使用说明

`competition/source/` 中的报名表可能包含个人信息。请只在获授权的竞赛协作场景中使用，不要把姓名、联系方式等内容复制到公开 Issue、Pull Request、截图或演示数据中。

开发和本地演示应以根 README、`source/` 当前源码、自动化测试及 `competition/final/` 最终文档为准。
