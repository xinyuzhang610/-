# 智教通文档中心

这里集中保存智教通的竞赛材料、产品设计、开发指南、质量记录与历史实施资料。产品代码的启动方式请先阅读仓库根目录的 [README](../README.md)。

## 文档导航

- [竞赛原始材料](competition/source/)：比赛通知和报名表等原始文件。
- [竞赛项目文档](competition/project/)：任务手册、需求与设计说明、操作手册、PPT 大纲和项目梳理。
- [设计规格](design/)：已确认的产品、视觉、交互与仓库治理设计。
- [开发指南](guides/)：环境配置和技术栈说明。
- [质量记录](quality/)：前端 QA 报告与验证截图。
- [历史归档](archive/)：已经执行或仅供追溯的实施计划和环境搭建报告。

## 资料使用说明

`competition/source/` 中的报名表可能包含个人信息。请只在获授权的竞赛协作场景中使用，不要把姓名、联系方式等内容复制到公开 Issue、Pull Request、截图或演示数据中。

`archive/` 中的文档记录了当时的实现过程，可能与当前代码存在差异。开发和部署应以根 README、当前源码及自动化测试为准。
