## 1. 引言

### 1.1 编写目的

本文档是"智教通——AI智能体教学辅助平台"的数据库设计说明书，旨在详细描述系统的数据库设计，包括数据库表结构、字段说明、索引设计、初始数据等，为开发人员提供数据库实现指导。

### 3.2 用户表 (users)

#### 3.2.1 表结构

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role ENUM('teacher', 'student', 'admin') NOT NULL DEFAULT 'teacher',
    name VARCHAR(50),
    school VARCHAR(100),
    subject VARCHAR(50),
    grade VARCHAR(20),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 3.2.2 字段说明

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|---------|------|--------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | - | 用户ID，自增主键 |
| username | VARCHAR(50) | UNIQUE, NOT NULL | - | 用户名，唯一索引 |
| password_hash | VARCHAR(255) | NOT NULL | - | 密码哈希值（bcrypt） |
| role | ENUM | NOT NULL | 'teacher' | 用户角色（teacher/student/admin） |
| name | VARCHAR(50) | | NULL | 真实姓名 |
| school | VARCHAR(100) | | NULL | 所在学校 |
| subject | VARCHAR(50) | | NULL | 教学科目 |
| grade | VARCHAR(20) | | NULL | 教学年级 |
| created_at | DATETIME | | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | | CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

#### 3.2.3 索引设计

| 索引名 | 索引类型 | 索引字段 | 说明 |
|--------|---------|---------|------|
| PRIMARY | 主键索引 | id | 主键 |
| idx_username | 唯一索引 | username | 用户名唯一 |
| idx_role | 普通索引 | role | 按角色查询 |

### 3.3 工具表 (tools)

#### 3.3.1 表结构

```sql
CREATE TABLE tools (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    icon VARCHAR(50) DEFAULT 'default',
    prompt_template TEXT NOT NULL,
    interface_config JSON,
    creator_id INT,
    is_preset BOOLEAN DEFAULT FALSE,
    is_public BOOLEAN DEFAULT TRUE,
    share_code VARCHAR(20) UNIQUE,
    usage_count INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (creator_id) REFERENCES users(id) ON DELETE SET NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 3.3.2 字段说明

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|---------|------|--------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | - | 工具ID |
| name | VARCHAR(100) | NOT NULL | - | 工具名称 |
| description | TEXT | | NULL | 工具描述 |
| category | ENUM | NOT NULL | - | 分类（文科/理科/通用） |
| subject | VARCHAR(50) | | NULL | 学科 |
| icon | VARCHAR(50) | | 'default' | 图标标识 |
| prompt_template | TEXT | NOT NULL | - | 系统提示词模板 |
| interface_config | JSON | | NULL | 界面配置 |
| creator_id | INT | FK | NULL | 创建者ID |
| is_preset | BOOLEAN | | FALSE | 是否预设工具 |
| is_public | BOOLEAN | | TRUE | 是否公开 |
| share_code | VARCHAR(20) | UNIQUE | NULL | 分享码 |
| usage_count | INT | | 0 | 使用次数 |
| created_at | DATETIME | | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | | CURRENT_TIMESTAMP ON UPDATE | 更新时间 |

#### 3.3.3 索引设计

| 索引名 | 索引类型 | 索引字段 | 说明 |
|--------|---------|---------|------|
| PRIMARY | 主键索引 | id | 主键 |
| idx_category | 普通索引 | category | 按分类查询 |
| idx_subject | 普通索引 | subject | 按学科查询 |

### 3.5 使用记录表 (usage_logs)

#### 3.5.1 表结构

```sql
CREATE TABLE usage_logs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    tool_id INT NOT NULL,
    input_text TEXT,
    output_text TEXT,
    session_id VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 3.5.2 字段说明

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|---------|------|--------|------|
| id | INT | PRIMARY KEY, AUTO_INCREMENT | - | 记录ID |
| user_id | INT | FK, NULLABLE | NULL | 用户ID（匿名用户为空） |
| tool_id | INT | FK, NOT NULL | - | 工具ID |
| input_text | TEXT | | NULL | 输入内容 |
| output_text | TEXT | | NULL | 输出内容 |
| session_id | VARCHAR(50) | | NULL | 会话ID |
| created_at | DATETIME | | CURRENT_TIMESTAMP | 使用时间 |

#### 3.5.3 索引设计

| 索引名 | 索引类型 | 索引字段 | 说明 |
|--------|---------|---------|------|
| PRIMARY | 主键索引 | id | 主键 |
| idx_user | 普通索引 | user_id | 按用户查询 |
| idx_tool | 普通索引 | tool_id | 按工具查询 |
| idx_session | 普通索引 | session_id | 按会话查询 |
| idx_created | 普通索引 | created_at | 按时间查询 |
| FOREIGN KEY | 外键索引 | user_id | 关联users表 |
| FOREIGN KEY | 外键索引 | tool_id | 关联tools表 |

### 3.6 收藏表 (favorites)

#### 3.6.1 表结构

```sql
CREATE TABLE favorites (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    tool_id INT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE,
    UNIQUE KEY uk_user_tool (user_id, tool_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 3.6.2 字段说明

| 字段名 | 数据类型 | 约束 | 默认值 | 说明 |
|--------|---------|------|--------|------|

| user_id | INT | FK, NOT NULL | - | 用户ID |
| tool_id | INT | FK, NOT NULL | - | 工具ID |
| created_at | DATETIME | | CURRENT_TIMESTAMP | 收藏时间 |

#### 3.6.3 索引设计

| 索引名 | 索引类型 | 索引字段 | 说明 |
|--------|---------|---------|------|
| PRIMARY | 主键索引 | id | 主键 |
| uk_user_tool | 唯一索引 | (user_id, tool_id) | 用户-工具唯一 |
| idx_user | 普通索引 | user_id | 按用户查询 |
| FOREIGN KEY | 外键索引 | user_id | 关联users表 |
| FOREIGN KEY | 外键索引 | tool_id | 关联tools表 |

## 5. 索引设计

### 5.1 索引策略

| 策略 | 说明 |
|------|------|
| 主键索引 | 每个表的主键自动创建索引 |
| 唯一索引 | 用户名、分享码等需要唯一性的字段 |
| 普通索引 | 常用查询字段，如分类、角色等 |

## 9. 附录

### 9.1 建表SQL汇总

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS zjtdb DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE zjtdb;

-- 用户表
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role ENUM('teacher', 'student', 'admin') NOT NULL DEFAULT 'teacher',
    name VARCHAR(50),
    school VARCHAR(100),
    subject VARCHAR(50),
    grade VARCHAR(20),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_username (username),
    INDEX idx_role (role)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 工具表
CREATE TABLE tools (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    icon VARCHAR(50) DEFAULT 'default',
    prompt_template TEXT NOT NULL,
    interface_config JSON,
    creator_id INT,
    is_preset BOOLEAN DEFAULT FALSE,
    is_public BOOLEAN DEFAULT TRUE,
    share_code VARCHAR(20) UNIQUE,
    usage_count INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (creator_id) REFERENCES users(id) ON DELETE SET NULL,
    INDEX idx_category (category),
    INDEX idx_subject (subject),
    INDEX idx_is_preset (is_preset),
    INDEX idx_share_code (share_code),
    INDEX idx_creator (creator_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 工具模板表
CREATE TABLE tool_templates (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    description TEXT,
    prompt_template TEXT NOT NULL,
    default_config JSON,
    icon VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_category (category)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 使用记录表
CREATE TABLE usage_logs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    tool_id INT NOT NULL,
    input_text TEXT,
    output_text TEXT,
    session_id VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE,
    INDEX idx_user (user_id),
    INDEX idx_tool (tool_id),
    INDEX idx_session (session_id),
    INDEX idx_created (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 收藏表
CREATE TABLE favorites (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    tool_id INT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE,
    UNIQUE KEY uk_user_tool (user_id, tool_id),
    INDEX idx_user (user_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 推荐规则表
CREATE TABLE recommend_rules (
    id INT PRIMARY KEY AUTO_INCREMENT,
    subject VARCHAR(50) NOT NULL,
    need_type VARCHAR(50) NOT NULL,
    tool_ids JSON NOT NULL,
    priority INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_subject_need (subject, need_type)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```
