## 1. 引言

### 1.1 编写目的

本文档是"智教通——AI智能体教学辅助平台"的详细设计说明书，旨在对系统的各个模块进行详细的设计，包括接口详细设计、数据库详细设计、界面详细设计、算法设计等，为开发人员提供具体的实现指导。

### 1.2 项目背景

本项目是湖南省第22届大学生计算机程序设计竞赛（Web应用开发类）参赛作品。系统面向教育场景，通过需求引导帮助教师发现教学需求、体验AI智能体工具，并引导其使用智能体平台创建自己的学习工具。

## 2. 数据库详细设计

### 2.1 数据库表结构（SQL DDL）

#### 2.1.1 用户表 (users)

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role ENUM('teacher', 'student', 'admin') NOT NULL DEFAULT 'teacher',
    name VARCHAR(50),
    school VARCHAR(100),
    subject VARCHAR(50),
    grade VARCHAR(20),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_username (username),
    INDEX idx_role (role)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

**字段说明：**

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | INT | PK, AUTO_INCREMENT | - | 用户ID，自增主键 |
| username | VARCHAR(50) | UNIQUE, NOT NULL | - | 用户名，唯一索引 |
| password_hash | VARCHAR(255) | NOT NULL | - | 密码哈希值 |
| role | ENUM | NOT NULL | 'teacher' | 用户角色 |
| name | VARCHAR(50) | | NULL | 真实姓名 |
| school | VARCHAR(100) | | NULL | 所在学校 |
| subject | VARCHAR(50) | | NULL | 教学科目 |
| grade | VARCHAR(20) | | NULL | 教学年级 |
| created_at | DATETIME | | CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | | CURRENT_TIMESTAMP | 更新时间 |

#### 2.1.2 工具表 (tools)

```sql
CREATE TABLE tools (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    icon VARCHAR(50) DEFAULT 'default',
    prompt_template TEXT NOT NULL,
    interface_config JSON,
    creator_id INT,
    is_preset BOOLEAN DEFAULT FALSE,
    is_public BOOLEAN DEFAULT TRUE,
    share_code VARCHAR(20) UNIQUE,
    usage_count INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (creator_id) REFERENCES users(id) ON DELETE SET NULL,
    INDEX idx_category (category),
    INDEX idx_subject (subject),
    INDEX idx_is_preset (is_preset),
    INDEX idx_share_code (share_code),
    INDEX idx_creator (creator_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

**字段说明：**

| 字段 | 类型 | 约束 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | INT | PK, AUTO_INCREMENT | - | 工具ID |
| name | VARCHAR(100) | NOT NULL | - | 工具名称 |
| description | TEXT | | NULL | 工具描述 |
| category | ENUM | NOT NULL | - | 分类（文科/理科/通用） |
| subject | VARCHAR(50) | | NULL | 学科 |
| icon | VARCHAR(50) | | 'default' | 图标标识 |
| prompt_template | TEXT | NOT NULL | - | 系统提示词模板 |
| interface_config | JSON | | NULL | 界面配置 |
| creator_id | INT | FK | NULL | 创建者ID |
| is_preset | BOOLEAN | | FALSE | 是否预设工具 |
| is_public | BOOLEAN | | TRUE | 是否公开 |
| share_code | VARCHAR(20) | UNIQUE | NULL | 分享码 |
| usage_count | INT | | 0 | 使用次数 |

#### 2.1.3 工具模板表 (tool_templates)

```sql
CREATE TABLE tool_templates (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    category ENUM('文科', '理科', '通用') NOT NULL,
    subject VARCHAR(50),
    description TEXT,
    prompt_template TEXT NOT NULL,
    default_config JSON,
    icon VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_category (category)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 2.1.4 使用记录表 (usage_logs)

```sql
CREATE TABLE usage_logs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    tool_id INT NOT NULL,
    input_text TEXT,
    output_text TEXT,
    session_id VARCHAR(50),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE SET NULL,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE,
    INDEX idx_user (user_id),
    INDEX idx_tool (tool_id),
    INDEX idx_session (session_id),
    INDEX idx_created (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 2.1.5 收藏表 (favorites)

```sql
CREATE TABLE favorites (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    tool_id INT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (tool_id) REFERENCES tools(id) ON DELETE CASCADE,
    UNIQUE KEY uk_user_tool (user_id, tool_id),
    INDEX idx_user (user_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 2.1.6 推荐规则表 (recommend_rules)

```sql
CREATE TABLE recommend_rules (
    id INT PRIMARY KEY AUTO_INCREMENT,
    subject VARCHAR(50) NOT NULL,
    need_type VARCHAR(50) NOT NULL,
    tool_ids JSON NOT NULL,
    priority INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_subject_need (subject, need_type)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

#### 5.1.1 算法描述

```python
def get_recommend_tools(category: str, need_type: str) -> List[Tool]:
    """
    根据学科分类和需求类型获取推荐工具

    参数：
    - category: 学科分类（文科/理科）
    - need_type: 需求类型（课前备课/课堂教学/课后辅导/学生兴趣激发）

    返回：
    - 推荐工具列表
    """
    # 1. 查询推荐规则
    rule = db.query(RecommendRule).filter(
        RecommendRule.subject == category,
        RecommendRule.need_type == need_type
    ).first()

    if not rule:
        # 默认返回该分类的所有工具
        return db.query(Tool).filter(
            Tool.category == category,
            Tool.is_preset == True
        ).all()

    # 2. 根据规则中的tool_ids查询工具
    tool_ids = rule.tool_ids  # JSON数组
    tools = db.query(Tool).filter(
        Tool.id.in_(tool_ids)
    ).all()

    return tools
```

```python
async def process_chat(tool_id: int, message: str, session_id: str) -> dict:
    """
    处理AI对话请求

    参数：
    - tool_id: 工具ID
    - message: 用户消息
    - session_id: 会话ID

    返回：
    - { reply: str, session_id: str }
    """
    # 1. 验证工具存在
    tool = db.query(Tool).filter(Tool.id == tool_id).first()
    if not tool:
        raise HTTPException(status_code=404, detail="工具不存在")

    # 2. 生成或获取session_id
    if not session_id:
        session_id = f"sess_{uuid.uuid4().hex[:12]}"

    # 3. 获取对话历史
    history = get_chat_history(session_id)

    # 4. 构建消息列表
    messages = [
        {"role": "system", "content": tool.prompt_template},
        *history,
        {"role": "user", "content": message}
    ]

    # 5. 调用DeepSeek API
    response = await call_deepseek_api(messages)

    # 6. 记录使用日志
    record_usage(tool_id, message, response, session_id)

    # 7. 更新工具使用次数
    update_tool_usage_count(tool_id)

    return {
        "reply": response,
        "session_id": session_id
    }

async def call_deepseek_api(messages: list) -> str:
    """
    调用DeepSeek API

    参数：
    - messages: 消息列表

    返回：
    - AI回复内容
    """
    import httpx

    async with httpx.AsyncClient() as client:
        response = await client.post(
            "https://api.deepseek.com/chat/completions",
            headers={
                "Authorization": f"Bearer {DEEPSEEK_API_KEY}",
                "Content-Type": "application/json"
            },
            json={
                "model": "dsv4",
                "messages": messages,
                "temperature": 0.7,
                "max_tokens": 2000
            },
            timeout=30.0
        )

        result = response.json()
        return result["choices"][0]["message"]["content"]
```

### 5.3 分享链接生成算法

#### 5.3.1 算法描述

```python
def generate_share_link(tool_id: int) -> dict:
    """
    生成工具分享链接

    参数：
    - tool_id: 工具ID

    返回：
    - { share_url: str, qr_code: str }
    """
    # 1. 获取工具
    tool = db.query(Tool).filter(Tool.id == tool_id).first()

    # 2. 生成分享码（如果不存在）
    if not tool.share_code:
        tool.share_code = generate_unique_code()
        db.commit()

    # 3. 拼接分享URL
    share_url = f"{BASE_URL}/tool/{tool.share_code}"

    # 4. 生成二维码
    qr_code = generate_qr_code(share_url)

    return {
        "share_url": share_url,
        "qr_code": qr_code  # Base64编码的图片
    }

def generate_unique_code() -> str:
    """
    生成唯一的分享码

    使用6位字母数字组合
    """
    import random
    import string

    while True:
        code = ''.join(random.choices(string.ascii_letters + string.digits, k=6))
        # 检查是否已存在
        exists = db.query(Tool).filter(Tool.share_code == code).first()
        if not exists:
            return code
```
