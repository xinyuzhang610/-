## 1. 系统概述

### 1.1 系统简介

"智教通"是一个面向教育场景的AI智能体教学辅助平台，通过**需求引导**帮助教师发现教学需求、体验AI智能体工具，并引导其使用智能体平台创建自己的学习工具。

### 1.2 主要功能

| 功能模块 | 功能说明 |
|---------|---------|
| 需求引导 | 通过多步骤交互帮助教师发现和明确教学需求 |
| AI智能问答 | 基于DeepSeek API的文本生成和多模态交互 |
| 工具广场 | 分类展示和浏览各类AI智能体工具 |
| 工具管理 | 工具的创建、编辑、分享和使用统计 |
| 数据看板 | 使用数据的可视化展示和分析 |
| 学习记录 | 学生的学习历史记录查看 |

### 1.3 用户角色

| 角色 | 说明 |
|------|------|
| 教师 | 使用平台创建/管理教学工具，查看学生数据 |
| 学生 | 使用工具进行学习，探索工具广场 |
| 管理员 | 管理平台内容、审核工具（后台） |

### 3.3 学生使用

**操作方式：**

- **方式一：通过分享链接访问**
  - 老师分享工具链接或二维码
  - 学生点击链接直接进入工具使用页面
  - 无需登录

- **方式二：访问工具广场**
  - 访问系统首页
  - 进入工具广场浏览工具
  - 选择工具进行使用

#### 4.2.2 创建新工具

**操作步骤：**

1. 点击"我的工具"页面右上角的"+ 创建新工具"
2. 选择创建方式：
   - AI对话创建
   - 平台创建工作流
3. 按照提示完成工具创建

**AI对话创建步骤：**

1. 点击"开始对话"
2. 描述你想要的工具（如"我想创建一个古诗词赏析工具"）
3. AI会自动生成工具配置
4. 确认配置后保存

#### 4.2.3 编辑工具

**操作步骤：**

1. 在工具列表中找到要编辑的工具
2. 点击"编辑"按钮
3. 修改工具的配置信息
4. 点击"保存"按钮

#### 4.2.4 生成分享链接

**操作步骤：**

1. 在工具列表中找到要分享的工具
2. 点击"生成分享链接"按钮
3. 系统生成分享链接和二维码
4. 复制链接或下载二维码分享给学生

**操作步骤：**

1. 在工具列表中找到要删除的工具
2. 点击"删除"按钮
3. 确认删除操作

## 6. 常见问题

### 6.1 登录相关

**Q：忘记密码怎么办？**

A：请联系系统管理员重置密码。

**Q：用户名被占用怎么办？**

A：请更换一个用户名重新注册。

### 6.2 工具使用相关

**Q：AI回复很慢怎么办？**

A：AI回复需要调用外部API，可能会有延迟，请耐心等待。

**Q：AI回复的内容不准确怎么办？**

A：AI生成的内容仅供参考，建议结合教材和老师指导使用。

**Q：如何收藏工具？**

A：在工具使用页面点击"收藏"按钮即可收藏。

### 6.3 工具管理相关

**Q：创建的工具别人能看到吗？**

A：默认创建的工具是公开的，其他人可以通过工具广场看到。如果不希望公开，可以在编辑时设置为私有。

**Q：分享链接有时效吗？**

A：分享链接默认永久有效，除非工具被删除。

### 6.4 技术问题

**Q：页面加载不出来怎么办？**

A：请检查网络连接，或尝试刷新页面。如果问题持续，请联系管理员。

**Q：浏览器提示不兼容怎么办？**

A：请使用Chrome、Firefox、Edge或Safari的最新版本访问。
