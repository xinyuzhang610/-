# 智教通 - AI智能体教学辅助平台

一个帮助教师发现教学需求并体验AI代理工具的教育类Web应用。

## 🌟 项目特色

- **智能教学需求发现** - AI自动分析教学场景，精准识别教师需求
- **AI代理工具体验** - 集成多种AI代理工具，支持智能备课、作业批改
- **海量教学资源库** - 整合优质教学资源，智能推荐适配内容
- **个性化学习路径** - 基于学生画像，智能规划个性化学习路径
- **精准学情分析** - 实时追踪学习进度，多维度分析学情数据

## 🎨 设计风格

采用小米MiMo的设计风格：
- 米棕色系配色方案
- 简洁现代的UI设计
- 流畅的动画效果
- 响应式布局，支持移动端

## 🚀 快速开始

1. 克隆项目
   ```bash
   git clone https://github.com/xinyuzhang610/-.git
   cd -
   ```

2. 打开首页
   ```bash
   # 直接在浏览器中打开
   open index.html
   ```

3. 替换素材
   - 查看 `素材清单.md` 了解需要补充的素材
   - 使用AI生成或搜索合适的图片
   - 替换 `assets/` 目录下的占位符图片

## 🎯 技术栈

- HTML5
- CSS3（CSS变量、Flexbox、Grid）
- JavaScript（ES6+）
- Canvas API（墨水特效）
- IntersectionObserver API（滚动动画）

## 📝 设计规范

### 颜色方案
- 背景色：`#fcfaf8`（米白色）

## 🔧 功能特性

1. **墨水揭示特效** - 鼠标移动时在Hero区域画布上"雕刻"墨水点，揭示底层背景图
2. **打字机效果** - 副标题和卡片标题的打字机动画
3. **平滑滚动** - 点击导航链接平滑滚动到对应区域
4. **响应式设计** - 完美适配桌面端和移动端
